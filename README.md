# 🤖 ClassifyIQ

> Upload Data. Train Models. Discover Intelligence.

ClassifyIQ is an AI-powered machine learning classification platform that enables users to upload datasets, analyze features, train classification models, compare performance metrics, and generate intelligent insights through an interactive dashboard.

## ✨ Features

- 📂 CSV Dataset Upload
- 📊 Dataset Explorer
- 🧠 Classification Model Training
- 📈 Performance Evaluation
- 🎯 Accuracy Comparison
- 🔍 Feature Importance Analysis
- 🤖 AI-Generated Insights
- 📉 Interactive Charts and Visualizations

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Machine Learning Concepts

## 🔬 Supported Models

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

## 📊 Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## 🎯 Project Goal

To simplify machine learning classification workflows through an intuitive interface that helps users understand datasets, compare models, and interpret results.
